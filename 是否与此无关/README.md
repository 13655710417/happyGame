# 是否与此无关 - 故事推理游戏

一个简约高级风格的单人互动游戏，类似于海龟汤（Lateral Thinking Puzzle）。玩家需要通过提问来还原完整故事。

## 游戏功能

1. **不完整的故事片段** - 游戏提供一个不完整的故事开头
2. **提问系统** - 玩家可以提出是/否问题，系统回答"是"、"否"或"与此无关"
3. **故事还原** - 玩家可以随时提交自己推理出的完整故事，系统会验证是否正确
4. **查看答案** - 玩家可以选择直接查看完整故事
5. **游戏统计** - 记录提问次数和游戏时间
6. **多个故事** - 内置多个不同的推理故事

## 游戏玩法

1. 阅读故事片段
2. 在输入框中提出是/否问题（例如："这个男人认识酒保吗？"）
3. 根据系统的回答继续推理
4. 当你认为已经推理出完整故事时，点击"还原故事"按钮提交
5. 如果故事大致正确，游戏胜利！
6. 任何时候可以点击"查看答案"直接查看完整故事

## 技术特点

- 响应式设计，支持桌面和移动设备
- 简约高级的UI设计，使用渐变色和优雅的字体
- 内置3个不同的推理故事
- 简单的关键词匹配验证系统
- 游戏状态管理和计时功能

## 运行游戏

游戏是纯前端应用，无需服务器即可运行：

1. 打开 `index.html` 文件在浏览器中
2. 或者使用本地HTTP服务器：
   ```bash
   # Python 3
   python -m http.server 8000

   # Node.js with http-server
   npx http-server
   ```

3. 在浏览器中访问 `http://localhost:8000`

## 自定义故事

要添加新的故事，编辑 `game.js` 文件中的 `GameState.stories` 数组：

```javascript
{
    id: 4,
    fragment: "故事片段文本",
    fullStory: "完整故事文本",
    keywords: ["关键词1", "关键词2", ...],
    questions: [
        { question: "预设问题1", answer: "是/否/与此无关" },
        { question: "预设问题2", answer: "是/否/与此无关" }
    ]
}
```
