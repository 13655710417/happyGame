# 智能聊天网站增强版

## 项目概述

这是一个经过全面增强的智能聊天机器人网站，整合了多种人工智能技术，提供更智能、更个性化的聊天体验。

## 主要改进

### 1. 自然语言处理增强
- **NLP模块** (`nlp-module.js`): 使用TensorFlow.js进行意图识别、实体提取和情感分析
- **智能分析**: 能够分析用户消息的意图、情绪、话题和关键信息
- **上下文理解**: 基于对话历史进行连贯回应

### 2. 机器学习对话生成
- **ML对话模块** (`ml-dialogue.js`): 基于神经网络的对话生成系统
- **在线学习**: 从对话历史中学习并改进回复质量
- **自适应生成**: 根据对话上下文和用户情绪调整回复风格

### 3. 动态知识库系统
- **知识管理器** (`knowledge-manager.js`): 可扩展的知识库管理系统
- **知识图谱**: 建立知识之间的关联关系
- **用户自定义**: 支持用户添加、修改、删除知识
- **智能检索**: 基于上下文和用户兴趣推荐相关知识

### 4. 多模态交互功能
- **多模态模块** (`multimodal-module.js`): 语音输入/输出和表情处理
- **语音识别**: 支持中文语音输入
- **语音合成**: 支持情感化语音输出
- **表情处理**: 智能添加和识别表情符号

### 5. 用户体验优化
- **性能监控**: 实时监控系统性能并自动优化
- **响应式设计**: 适配各种屏幕尺寸
- **主题切换**: 支持浅色/深色主题
- **实时分析**: 输入时实时分析用户意图

### 6. 系统集成
- **最终增强版** (`script-final-enhanced.js`): 整合所有模块的完整系统
- **统一状态管理**: 集中管理所有对话状态和用户数据
- **智能决策**: 根据上下文选择最佳回复策略

## 使用方法

### 1. 基本聊天
1. 打开 `index.html`
2. 在输入框中输入消息
3. 按Enter或点击发送按钮
4. AI会智能回复你的消息

### 2. 语音功能
- 点击麦克风按钮开启语音输入
- 系统会自动识别你的语音并转换为文字
- 支持语音输出（需在设置中开启）

### 3. 知识库管理
1. 打开 `knowledge-ui.html`
2. 可以添加、修改、删除知识
3. 可以导入/导出知识库
4. 可以搜索和浏览知识

### 4. 高级功能
- **Ctrl+Shift+S**: 查看系统状态（开发者工具）
- **表情选择器**: 点击表情按钮选择表情
- **主题切换**: 点击调色板按钮切换主题

## 技术特性

### 智能特性
- **情感分析**: 准确识别用户情绪并给予恰当回应
- **意图识别**: 理解用户意图并提供相关回复

## 开发说明

### 模块扩展
每个模块都可以独立扩展：

1. **NLP模块**: 可以添加更多语言支持或改进分析算法
2. **ML模块**: 可以训练更复杂的模型或添加新的训练数据
3. **知识库**: 可以添加新的知识分类或改进检索算法
4. **多模态**: 可以添加图像识别或其他交互方式

### 配置选项
在 `script-final-enhanced.js` 中可以配置：

```javascript
this.config = {
    aiName: '小韬',              // AI名称
    personality: 'gentle',       // AI性格
    soundEnabled: true,          // 音效开关
    useVoiceInput: false,        // 语音输入开关
    useVoiceOutput: false,       // 语音输出开关
    useMLGeneration: true,       // 机器学习生成开关
    theme: 'light'               // 主题
};
```

### 数据存储
系统使用LocalStorage存储以下数据：
- 用户配置
- 用户档案
- 对话历史
- 知识库数据
- 机器学习模型权重
