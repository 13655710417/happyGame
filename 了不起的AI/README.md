# 智能治愈系AI聊天平台

一个粉色治愈风格的智能AI聊天伴侣，采用增强版模拟AI回复技术，无需网络即可使用。具备对话记忆、情感分析、个性化学习和知识库功能。

## 🚀 智能增强功能

### 🧠 对话记忆与连贯性
- **短期记忆**：记住最近10轮对话内容
- **上下文理解**：基于之前的对话进行连贯回应
- **话题跟踪**：识别和跟踪当前讨论话题
- **跟进提问**：智能提出跟进问题，深化对话

### 😊 情感分析与支持
- **情绪识别**：自动检测开心、难过、生气、疲惫、焦虑等情绪
- **情感回应**：根据用户情绪给予恰当的情感支持
- **情绪状态跟踪**：持续跟踪用户情绪变化

### 📚 个性化记忆与学习
- **用户档案**：记住用户的姓名、年龄、地点、职业等信息
- **兴趣偏好**：学习用户的爱好、喜欢和不喜欢的事物
- **对话历史**：保存重要对话内容，提供个性化回应
- **本地存储**：所有个人信息仅保存在您的设备中

### 💡 知识库与建议
- **科学知识**：自然、科学、历史等领域的趣味知识
- **实用建议**：压力管理、学习技巧、健康生活等实用建议
- **趣味内容**：笑话、冷知识、名言警句
- **智能匹配**：根据对话上下文提供相关知识

### ⚙️ 基础功能
- **治愈系设计**：粉色柔和配色，圆角元素，动画效果
- **AI伙伴**：初始名称为"小韬"，可自定义名称和性格
- **多种性格**：温柔治愈型、活泼开朗型、智慧导师型、知心朋友型
- **完整功能**：
  - 实时聊天对话
  - 快速回复建议
  - 聊天记录管理
  - 个性化设置
  - 本地存储保存设置

## 🎯 使用说明

### 快速开始
1. **打开应用**：双击 `index.html` 在浏览器中打开
2. **开始聊天**：在下方输入框输入消息，按Enter或点击发送按钮
3. **使用快速回复**：点击上方的快速回复按钮发送常用消息
4. **个性化设置**：点击左侧菜单的"设置"按钮更改AI名称、性格等
5. **管理聊天**：使用"新对话"或"清空记录"管理聊天历史

### 智能功能体验
尝试以下对话，体验智能功能：
- **情感识别**："我今天很难过" → AI识别情绪并安慰
- **知识问答**："什么是光合作用？" → 提供科学知识
- **建议咨询**："学习压力大怎么办" → 给出实用建议
- **个性化**："我叫小明，喜欢看电影" → AI记住你的名字和爱好
- **连贯对话**：连续谈论同一话题，AI会记住上下文

### 隐私保护
- 所有数据保存在本地浏览器中
- 清空浏览器数据会丢失所有记录
- 无网络连接，完全离线运行

## 🔧 技术实现

### 智能系统架构
- **多层响应系统**：情感回应 → 话题回应 → 知识回应 → 个性化回应
- **上下文管理器**：跟踪对话状态、话题、情绪和用户信息
- **知识库引擎**：结构化知识存储和智能检索
- **用户档案系统**：本地存储用户信息和学习数据

### 核心技术
- **纯前端实现**：无需后端服务器，完全离线运行
- **高级LocalStorage**：存储用户设置、对话记忆、用户档案
- **响应式设计**：完美适配手机、平板和电脑
- **现代Web技术**：使用ES6类、模板字符串、Set/Map数据结构
- **隐私保护**：所有数据保存在用户设备，无网络传输

### 算法特性
- **情感分析算法**：基于关键词的情绪状态识别
- **话题分类算法**：多话题识别和分类系统
- **知识匹配算法**：上下文相关的知识推荐
- **个性化算法**：基于用户档案的个性化回复生成

## 自定义设置

- **AI名称**：可更改为任意喜欢的名字
- **AI性格**：四种不同性格可选
- **提示音**：开启或关闭消息提示音
- **自动滚动**：开启或关闭自动滚动到最新消息

## 浏览器兼容性

支持所有现代浏览器（Chrome、Firefox、Safari、Edge）。

## 注意事项

- 所有数据仅保存在本地浏览器中，清空浏览器数据会丢失设置和记录
- AI回复为模拟生成，基于预设模板和关键词匹配
- 如需更智能的对话，可考虑集成真实AI API
