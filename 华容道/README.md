# 数字华容道游戏

一个复古风格的数字华容道游戏，使用纯HTML、CSS和JavaScript构建。

## 功能特性

- **多种规格**: 支持4×4、5×5、6×6、7×7四种棋盘尺寸
- **复古风格**: 怀旧的配色和视觉设计
- **游戏统计**: 实时显示步数和用时
- **获胜提示**: 成功复原后显示恭喜弹窗
- **游戏教程**: 内置游戏技巧和策略教程
- **提示功能**: 高亮显示可移动的方块
- **响应式设计**: 适配各种屏幕尺寸

## 如何运行

1. 双击 `index.html` 文件在浏览器中打开
2. 或者将文件夹复制到Web服务器目录中

## 游戏玩法

1. **选择规格**: 点击上方的规格按钮选择棋盘大小
2. **开始游戏**: 点击"开始新游戏"按钮
3. **移动方块**: 点击与空白位置相邻的数字方块进行移动
4. **游戏目标**: 将数字按顺序排列（从左到右，从上到下），最后一个位置为空

## 游戏技巧

详细技巧请点击"游戏教程"按钮查看。

## 文件结构

- `index.html` - 主HTML文件
- `style.css` - 样式文件（复古风格）
- `script.js` - 游戏逻辑JavaScript文件
- `README.md` - 说明文件

## 技术细节

- 使用纯JavaScript实现游戏逻辑
- CSS Grid布局实现响应式棋盘
- 使用Font Awesome图标
- 使用Google Fonts中文字体
- 完全在客户端运行，无需后端
