# 简约数独游戏

一个设计简约大气的数独游戏，包含经典模式和连续数独变体。

## 功能特性

### 游戏模式
1. **经典数独**
   - 三种难度：中等（约35个数字）、困难（约25个数字）、大师（需假设推理）
   - 标准数独规则：每行、每列、每宫包含1-9不重复

2. **连续数独**（特殊变体）
   - 两种难度：困难、大师
   - 额外规则：相邻单元格间的数字必须连续（差值为1）
   - 灰色粗线标记表示连续性

### 游戏功能
- 响应式界面，支持桌面和移动设备
- 键盘支持（数字键1-9，方向键移动）
- 提示系统（每个游戏3次提示）
- 实时错误检查
- 计时器
- 完整求解功能
- 详细的游戏规则说明

## 游戏规则详解

### 经典数独
- 目标：填充9×9网格，使每行、每列、每个3×3宫都包含数字1-9且不重复
- 难度说明：
  - **中等**：约35个已知数字，适合初学者
  - **困难**：约25个已知数字，需要逻辑推理
  - **大师**：需假设推理，难度在困难与大师之间

### 连续数独
- **特殊规则**：相邻单元格（上下左右）如果有灰色粗线标记，则数字必须连续（差值为1）
- **灰色粗线**：表示相邻两个数字必须是连续的（如1-2或2-1）
- **没有灰色粗线**：相邻单元格之间没有连续性要求
- **示例**：
  - 两个相邻单元格之间有灰色粗线：它们的数字差必须为1
  - 如1-2, 2-3, 3-4等连续数字对

## 操作指南
1. **选择模式**：点击顶部按钮选择经典模式或连续数独
2. **选择难度**：根据模式选择相应难度等级
3. **开始游戏**：点击"新游戏"按钮生成谜题
4. **填写数字**：
   - 点击单元格选中，然后点击底部数字按钮
   - 或使用键盘数字键1-9
   - 按0、退格键或删除键清除数字
5. **使用功能**：
   - **提示**：显示一个正确数字（每局3次）
   - **检查**：验证当前填写的数字是否正确
   - **求解**：显示完整解决方案

## 设计特点

- **直观交互**：选中单元格高亮相关行、列、宫
- **响应式布局**：自适应不同屏幕尺寸
- **无障碍设计**：支持键盘操作和屏幕阅读器

## 注意事项
1. 连续数独的实现为简化版本，实际约束生成可能不够完善
2. 大师难度可能需要"假设推理"，即尝试性填写并验证
3. 游戏数据保存在浏览器内存中，刷新页面会重置
