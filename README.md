 tao出个游戏 - 动漫风格小游戏平台

一个集合了四款有趣小游戏的动漫风格平台，提供轻松愉快的游戏体验。

 🎮 平台特色

### ✨ 核心功能
- 动漫风格界面 - 可爱的动漫角色和色彩设计
- 四款精选游戏 - 推理、逻辑、休闲、益智全覆盖
- 响应式设计 - 支持电脑、平板、手机全平台
- 留言互动系统 - 玩家可以与开发者直接交流
- 主题切换 - 支持亮色/暗色主题模式

### 🎯 游戏列表
1. 是否与此无关 - 海龟汤风格推理游戏
2. 数独 - 经典数独与连续数独变体
3. 羊了个羊 - 羊羊消除休闲游戏
4. 一笔画 - 一笔连接所有点的益智游戏
5. 华容道 - 经典数字华容道游戏

## 💡 使用说明

### 游戏入口
1. 打开平台首页
2. 在"游戏大厅"区域点击任意游戏卡片
3. 点击"开始游戏"按钮进入对应游戏

### 留言功能
1. 滚动到页面底部的"留言板"区域
2. 填写昵称、游戏建议和详细描述
3. 点击"发送留言"提交
4. 留言将显示在右侧的留言列表中

### 主题切换
- 点击右上角的月亮图标切换到暗色主题
- 再次点击太阳图标切换回亮色主题
- 主题设置会自动保存

## 🛠️ 开发说明

### 添加新游戏
1. 在对应目录创建游戏文件
2. 确保游戏入口为 `index.html`
3. 在平台首页的 `index.html` 中添加游戏卡片
4. 更新游戏卡片链接到正确的路径

### 样式定制
- 修改 `style.css` 中的 CSS 变量来调整颜色主题
- 更新 `:root` 中的颜色变量
- 调整动画时长和效果

### 功能扩展
- 在 `script.js` 中添加新的交互功能
- 扩展留言系统（如添加回复功能）
- 添加游戏评分系统
- 集成更多社交分享功能

## 📊 性能优化
